# Summarise bootstrap estimates

Summarise bootstrap estimates

## Usage

``` r
summarise_pls_bootstrap(boot_result)
```

## Arguments

- boot_result:

  Result returned by
  [`pls_bootstrap()`](https://fbertran.github.io/bigPLSR/reference/pls_bootstrap.md).

## Value

A data frame containing mean, standard deviation, percentile and BCa
confidence intervals for each coefficient.
